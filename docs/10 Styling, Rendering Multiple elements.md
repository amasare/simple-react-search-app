# Styling, Rendering Multiple Elements
Use map to easily iterate over result elements and return valid JSX.
Use searchComponent.scss class provided in the helper folder. Notice the scss file is ignored initially.

### Install some more webpack loaders to be able to understand and reduce css into bundle.js
```
npm install --save style-loader css-loader sass-loader node-sass
```

### Add webpack config for scss files
```
      {
          test: /\.scss$/,
          use: [{
              loader: "style-loader" // creates style nodes from JS strings
          }, {
              loader: "css-loader" // translates CSS into CommonJS
          }, {
              loader: "sass-loader" // compiles Sass to CSS
          }]
       }
```

Restart server to see changes.
