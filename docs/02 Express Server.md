# Creating Express Server With Static Page
At the end of this section, you should be able to run your server locally and see a "Hello World" page when you navigate to the listening host/port

### Install express server
```
npm install --save express
```

Notice package-lock.json is created

### Create server.js

### Add babel
```
npm install --save babel-cli babel-preset-env babel-loader
```

### Add webpack config
```
npm install --save webpack webpack-dev-middleware
```

# New Concepts
Webpack compiles multiple input files into one single file
Babel compiles next-gen JavaScript into browser-compatible JavaScript
Express is a fast, unopinionated, minimalist webframework for Node.js
